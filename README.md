# MovieApp

This project is a project made for the internship period of adesso company.

Technologies That Used:

+ Splash Screen
+ Hilt
+ Data Binding
+ Navigation
+ RecyclerView
+ ViewPager
+ Retrofit
+ Lottie
+ Moshi
+ MVI + MVVM
+ StateFlow
+ LiveData
+ Detekt
+ Coil
+ BottomNavigationBar
